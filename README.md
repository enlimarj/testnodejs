# INSTRUÇÕES PARA INSTALAÇÃO

## REQUISITOS

Instalar:
- Docker Community Edition (CE) ou Enterprise Edition (EE)
- Docker Compose 

## REQUISITOS PARA USO

- Criar uma nova rede chamada NETAPP

```
docker network create netapp
```

## EXECUTANDO CONTAINER

```bash
 nodejs$ docker-compose up -d
```
